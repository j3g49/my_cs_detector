Instructions to run the program:
  
1. Double click on the main() program
2. Double-check if the parameters bounds for finding code smells are within your acceptable range. Below are the three parameters.
		a. max_lines_class_method = maximum # of lines of code that a class or method can be allowed to have before we call it as a violation 
		b. max_params_method = maximum # of arguments that a method or function can have
		c. min_comment_words = Minimum # words needed for a comment to be considered as meaningful.
3. Enter the directory on which the code lynting needs to happen for code smell detection in 'dir'
4. Click on Run/execute button. The program will print the corresponding output in the console.

Note: comments are excluded in the consideration of # of lines of code.
